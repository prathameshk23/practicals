To run the files from string and collection folder do

```
java filename.java
```

To run the files from jdbc start a databse change the table and database name according to you and then run by doing 
for linux
```
java -cp .:mysql-connector-j-9.5.0.jar filename.java
```
for windows
```
java -cp .;mysql-connector-j-9.5.0.jar filename.java
```
